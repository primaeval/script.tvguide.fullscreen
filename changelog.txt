- see Help in Settings for Command Keys
- "Import addons.ini" will overwrite addon folders on next update timer
- use pip theme with 8 channels per page to match original tv guide icon layout
- click to load in addon browse dialog
- right click to add channels to categories
- reload all folders in settings
- use Spoyser's repo for Super Favourites or MySearch (now at DixieDean's repo)
